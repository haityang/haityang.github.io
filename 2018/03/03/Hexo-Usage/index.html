Hexo Usage

2018/03/03

Hexo使用方法

https://hexo.io/docs/

hexo常用命令

安装环境

sudo npm install -g hexo-cli

初始化工程

$ hexo init blog
$ cd blog
$ npm install

启动服务

hexo server

新加贴子

hexo new “Hexo Usage” //会在source/_posts下生成新文件
hexo generate

发布到github

安装 hexo-deployer-git，否则会报 ERROR Deployer not found: git 的错误。

npm install hexo-deployer-git –save

因为ssh上传，需要在git上传公钥，然后上传公程

hexo deploy

Hexo更换主题

1.修改_config.yml文件，更改theme:AlphaDust,该配置文件中有主题下载地址

2.从github上下载主题到themes/文件夹下，把主题名改为AlphaDust
